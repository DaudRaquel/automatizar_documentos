# 📄 Automação de Documentos — TOTVS RM

> **Python · Oracle · Selenium · ReportLab**

Sistema para geração automática de PDFs para motoristas a partir do ERP TOTVS RM (Oracle). Reduziu um processo que levava horas para minutos, eliminando erros manuais.

## ✨ Funcionalidades

- Extração de dados diretamente do Oracle (TOTVS RM)
- Geração de PDFs profissionais com ReportLab
- Automação Selenium para navegação no ERP
- Processamento em lote (múltiplos documentos de uma vez)
- Relatório de execução com status por documento

## 🛠️ Stack

```
Linguagem: Python
Banco:     Oracle (TOTVS RM)
PDF:       ReportLab
Automação: Selenium
```

## 📊 Impacto

- ⏱️ Redução de horas para minutos no processo
- ✅ Eliminação de erros manuais
- 📄 Centenas de PDFs gerados automaticamente
